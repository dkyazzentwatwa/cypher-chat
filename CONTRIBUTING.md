# Contributing

Thank you for considering contributing to **cypher-chat**!

## Development workflow
1. Fork and clone the repository.
2. Make your changes in a branch on your fork.
3. Run the build to ensure the firmware compiles:
   ```
   arduino-cli compile --fqbn esp32:esp32:esp32 cypher-chat.ino
   ```
4. Open a pull request with a clear description of your changes.

## Coding guidelines
- Use two spaces for indentation in C++ and Arduino files.
- Keep functions small and focused on a single task.
- Prefer descriptive variable names over comments.

## Reporting issues
Please include as much detail as possible—board type, steps to reproduce, and any logs—when filing an issue.

We welcome contributions of all sizes, from documentation fixes to new features.
