# Starbeam Student Worksheets

Use these pages in class or for self study.

**Worksheet 1: Build checklist**
- ESP32 power confirmed
- OLED shows status screen
- Buttons respond to short press
- Emergency broadcast works
- Device shows at least one peer

**Worksheet 2: Message flow**
1. Draw four nodes in a line.
2. Set TTL to 3 at the sender.
3. Mark the TTL at each hop.
4. Which node is the last to receive the message?

**Worksheet 3: Range testing**
- Test location
- Node spacing
- Obstacles present
- Max hops achieved
- Notes

**Worksheet 4: Security scenario**
- What happens if a device is stolen?
- How can a shared passkey be rotated?
- What should be done after a compromise?

**Worksheet 5: Emergency protocol**
- Who can initiate emergency broadcasts?
- How do you prevent false alarms?
- What is the cancel procedure?

**Worksheet 6: Design challenge**
- Add a feature that improves battery life.
- Add a feature that improves message clarity.
- Add a feature that improves trust between nodes.
