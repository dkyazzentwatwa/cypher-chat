# Starbeam Quick Cards

Print or copy these to each kit.

**Buttons**
- Button 3 short: cycle screens
- Button 3 long: emergency broadcast
- Button 1 long: cancel emergency

**Terminal**
- `status`
- `peers`
- `mesh`
- `emergency`
- `cancel`
- `send <1-4>`

**LED colors**
- Blue: idle
- Yellow: connecting
- Green: connected
- Red: emergency

**Emergency**
1. Confirm incident
2. Broadcast once
3. Await ACK
4. Cancel when resolved
