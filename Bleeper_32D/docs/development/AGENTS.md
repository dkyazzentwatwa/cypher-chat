# Agent Instructions

These instructions apply to the entire repository.

- **Style**: Use two spaces for indentation in all C++/Arduino and Markdown files.
- **Testing**: Before committing, run:
  ```bash
  arduino-cli compile --fqbn esp32:esp32:esp32 Bleeper.ino
  ```
  Ensure the command completes successfully.
- **Documentation**: Update relevant sections in `README.md` when altering features or behavior.
